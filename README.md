# Desert Rider
Desert Rider is a game about reckless driving. Your goal is to drive on the wrong side of the road as your score builds up. 
Avoid cars and listen to The Doors.

OBS! Your score is not displayed while driving.
  
### Controls
W and S to turn your car.
Space to accelerate.
Shift to switch music.

### Run Instructions

```python
  python Desert_Rider.py
```
